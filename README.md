# Vue Material

Project to implement MDC components ( https://material.io/develop/web/ ) with Vue2.

Now implemented:
- Top App Toolbar
- Drawer

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
